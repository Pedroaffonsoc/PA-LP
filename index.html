Projetos Futuros

Um protótipo de triagem jurídica que testei para um escritório de advocacia. À esquerda, a conversa que o cliente vê. À direita, o que a IA extrai dela, em tempo real.

Como funciona a triagem por IA

O cliente responde três perguntas: nome, WhatsApp e um relato livre do que aconteceu. Sem menu de opções, sem etapa de "IA analisando" no meio da conversa.

O relato vai para a API da Claude, que devolve um JSON estruturado: área jurídica principal e secundária, resumo dos fatos, parte contrária, prazo, valor do prejuízo, grau de urgência e as perguntas que ainda faltam para fechar o caso. Se algum dado não aparece no relato, o campo fica marcado como "não informado", nunca preenchido por suposição. O advogado recebe isso pronto. O cliente nunca vê essa parte.

Triagem para Assistências Técnicas

O mesmo conceito da triagem jurídica, agora para lojas de assistência técnica. O cliente chega pelo WhatsApp, responde às perguntas e a loja já sabe modelo, problema e contato — sem esforço humano na primeira triagem.

Como funciona o fluxo

O bot coleta o nome do cliente, o modelo do aparelho (iPhone, iPad, MacBook, Apple Watch etc.) e o tipo de problema (tela, bateria, iCloud, conector, contato com líquido ou outros). Para cada combinação, já existe uma lógica de roteamento diferente.

Para troca de tela, o bot identifica o modelo exato e entrega uma estimativa de preço em tempo real. Para bateria, iCloud ou aparelho morto, encaminha direto para o atendente com o diagnóstico já estruturado. O técnico recebe o JSON completo com tudo que precisa para dar um orçamento.

📋 Fluxo do Typebot
1
Boas-vindas — Bot se apresenta e pede o nome do cliente
2
Modelo do aparelho — iPhone, iPad, MacBook, Apple Watch ou Android
3
Tipo de problema — Tela, bateria, iCloud, conector, líquido ou outros
4
Roteamento inteligente — Para tela: seleciona modelo → entrega orçamento estimado
5
Coleta do WhatsApp — Número com DDD para o técnico fazer o retorno
6
Confirmação — Técnicos notificados. JSON estruturado entregue internamente
